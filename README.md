# 🌉 Bridge Rang (브릿지랑)

**Google Sheets 기반 엑셀 데이터 자동 분류 자동화 솔루션 (v2.10)**

Bridge Rang은 물류 및 유통 현장에서 발생하는 대량의 배송/주문 데이터를 각 목적지(창고)의 기준에 맞춰 자동으로 분류해 주는 윈도우용 데스크톱 애플리케이션입니다. 구글 스프레드시트와 실시간으로 연동하여 항상 최신 분류 기준을 유지할 수 있습니다.

---

## ✨ 주요 기능 (Features)

* **실시간 마스터 데이터 연동:** 구글 스프레드시트에 등록된 목적지별 고유 번호를 실시간으로 읽어와 분류 기준으로 삼습니다.
* **멀티 채널 자동 분류:** 입력된 원본 엑셀 데이터를 대조하여, 각 창고별 시트와 미분류 시트로 완벽하게 분할하여 새로운 엑셀 파일로 자동 생성합니다.
* **직관적인 GUI 환경:** 엑셀 파일 첨부뿐만 아니라 텍스트 직접 붙여넣기(우클릭 지원)를 지원하여 작업 속도를 극대화했습니다.
* **보안 및 편의성 강화:** 사용자의 구글 서비스 계정 키(JSON)를 텍스트 형태로 간편하게 등록할 수 있으며, 설정 정보는 윈도우 시스템 폴더(AppData)에 안전하게 숨김 보관되어 바탕화면을 깔끔하게 유지합니다.
* **작업 결과 요약:** 데이터 분류 완료 후, 각 창고별로 할당된 데이터 건수를 로그창에 즉시 요약하여 보여줍니다.

---

## 📦 '창고(Warehouse)' 란?

본 프로그램에서 '창고'는 데이터를 분류하는 **기준점이 되는 구글 스프레드시트 주소**를 의미합니다.
1. 각 창고(스프레드시트)의 **E열(5번째 열)**에는 해당 지역/창고로 배정되어야 할 고유 번호들이 나열되어 있어야 합니다.
2. 프로그램은 사용자가 입력한 원본 데이터의 특정 열 값과 이 창고 시트들의 번호를 대조합니다.
3. 조건이 일치하는 데이터는 해당 창고 번호가 부여된 엑셀 시트로 자동 분류됩니다.

## 🚀 설치 및 사용 방법 (How to Use)

### 1. 사전 준비 (인증 키)
* 구글 클라우드 콘솔(Google Cloud Console)에서 `Google Sheets API`와 `Google Drive API`를 활성화합니다.
* 서비스 계정을 생성하고 **JSON 형식의 키**를 발급받습니다.
* 기준이 될 구글 스프레드시트의 공유 설정에서 발급받은 서비스 계정(이메일)을 **편집자**로 추가합니다.

### 2. 프로그램 실행
* `BridgeRang_v2.10.exe` 실행 파일을 구동합니다. (또는 `python gui_main.py` 실행)
* 상단의 **[JSON 키 설정]** 버튼을 눌러 발급받은 키 파일이나 텍스트를 등록합니다.
* 엑셀 데이터를 분류할 **창고 주소(구글 스프레드시트 URL)**를 리스트에 추가합니다.
* 원본 데이터를 엑셀 파일로 불러오거나 빈칸에 직접 붙여넣고 **[분류 시작]**을 클릭합니다.
* 작업 완료 후 바탕화면의 `브릿지랑` 폴더에 통합 결과 엑셀 파일이 생성됩니다.

## 🛠️ 개발 환경 및 요구 사항 (Requirements)

* Python 3.8+
* `pandas` : 엑셀 데이터 프레임 처리
* `gspread` : 구글 스프레드시트 API 연동
* `google-auth` : 구글 서비스 계정 인증
* `openpyxl` : 엑셀(.xlsx) 파일 저장 엔진
* `tkinter` : GUI 구성 (Python 기본 내장)
