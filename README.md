# Bridge Rang (브릿지랑) - 택배 송장 자동 분류기

구글 스프레드시트와 연동하여 택배 송장 엑셀 파일을 자동으로 분류하고 정리해주는 프로그램입니다.
각 창고별 재고 데이터를 실시간으로 비교하여 출고 가능한 창고를 자동으로 지정합니다.

## 주요 기능

*   **자동 분류**: 업로드한 엑셀 파일의 주문 데이터를 분석하여 1창고, 2창고, 3창고 등으로 자동 분류합니다.
*   **실시간 재고 연동**: 구글 스프레드시트 API를 통해 최신 재고 현황을 반영합니다.
*   **다중 창고 지원**: 설정 메뉴에서 창고 URL을 자유롭게 추가/삭제할 수 있습니다.
*   **결과 파일 생성**: 분류된 결과를 창고별 개별 파일 및 통합 파일로 자동 저장합니다.
*   **보안**: 구글 서비스 계정 인증 키(JSON)를 사용하여 안전하게 연동됩니다.

## 설치 및 실행 방법

### 1. 실행 파일 사용 (Windows)
`dist/BridgeRang_v2.9.exe` 파일을 다운로드하여 실행하면 별도의 파이썬 설치 없이 바로 사용할 수 있습니다.

### 2. 소스 코드 실행
파이썬 환경에서 직접 실행하려면 아래 단계를 따르세요.

1.  **필수 패키지 설치**
    ```bash
    pip install -r requirements.txt
    ```

2.  **프로그램 실행**
    ```bash
    python gui_main.py
    ```

## 설정 방법

1.  **구글 인증 키 준비**: 구글 클라우드 콘솔에서 서비스 계정을 생성하고 JSON 키 파일을 다운로드합니다.
2.  **인증 키 등록**: 프로그램 우측 상단의 `⚙️ 설정` 버튼을 눌러 다운로드한 JSON 파일을 선택합니다.
3.  **스프레드시트 URL 등록**: 분류 기준이 되는 창고별 구글 스프레드시트 URL을 입력합니다. (`+` 버튼으로 추가 가능)
4.  **권한 설정**: 각 스프레드시트에 서비스 계정 이메일(JSON 파일 내 `client_email`)을 '뷰어' 권한으로 공유해야 합니다.
